## 📝 Description du projet
Mise en œuvre d'un système embarqué basé sur un microcontrôleur **ESP32** permettant la remontée de données de télémétrie. Le système lit les données d'un capteur de température et d'humidité (DHT22) et déclenche des alertes configurées.

## 🛠️ Technologies & Matériel
* **Microcontrôleur :** ESP32
* **Langage :** MicroPython
* **Capteurs/Actionneurs :** Capteur DHT22, LEDs d'alerte
* **Environnement de test :** Wokwi IoT Simulator

## ⚙️ Fonctionnement
1. Acquisition des données de température et d'humidité via le capteur DHT22.
2. Traitement des données via MicroPython.
3. Déclenchement d'une alerte visuelle (LED) si les seuils critiques sont dépassés.
